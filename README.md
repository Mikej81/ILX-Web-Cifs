# ILXWebCifs
##iRuleLX Webified CIFS Project

F5 BIG-IP iRulesLX (Unsupported) Project for Webified CIFS Shares.

![alt tag](http://i.imgur.com/7oO4D8xm.png)

Currently supports SMB2:ReadDir, SMB2:ReadFile, SMB2:WriteFile
